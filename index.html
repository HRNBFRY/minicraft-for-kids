メニュー

MiniCraft

クリックしてスタート
WASD: 移動　/　マウス: 視点　/　Space: ジャンプ（×2で飛行）
飛行中 Space・Shift: 上昇・下降　/　左クリック: 破壊　/　右クリック(長押しで連続設置): 設置
ホイール: ブロック切替　/　E: インベントリ　/　F: 影MOD　/　F5: 視点切替　/　Esc: マウス解除
🎮 Proコン(USB・Switch版マイクラと同じ配置): 左スティック移動・右スティック視点・B: ジャンプ（×2で飛行・Aで下降）
ZR: 破壊　/　ZL: 設置（長押しで連続設置）　/　L・R: ブロック切替（同時押しでスクリーンショット）　/　X・Y: インベントリ
左スティック押込み: 影MOD（独自機能）　/　－: 視点切替（独自機能）　/　＋: スタート・ポーズ
ネザーへ: 黒曜石で枠(4×5)→内側に火打石　エンドへ: 3×3の周りにフレーム12個＋エンダーアイ
作業台: 右クリックでクラフト画面　/　自動セーブ（localStorage）
右上の ☰ メニュー: 描画範囲（近・中・遠）切替　/　視点切替（一人称・三人称）　/　トップ画面に戻る
タッチ操作（iPad等）: 左下スティックで移動・画面右側ドラッグで視点・下部ボタンで操作（設置は長押しで連続設置）
